# Pagefolio PDF

Chromeで開いているタブをPDFとして保存するためのChrome拡張機能です。

## できること

- 現在のタブをPDFとして保存
- A4 / Letter / Legal の切り替え
- 縦 / 横の切り替え
- ページ区切り / 1枚の長いページの切り替え
- タイトルとURLの表示切り替え
- 保存時に場所を選ぶ / 自動保存の切り替え
- 背景色と画像を含める設定
- 画面表示に近いPDF / 印刷用CSSのPDFの切り替え
- 長いページを1ページPDFに収めるための自動縮小
- 1ページPDFで末尾が2ページ目に分かれないようにする余白補正
- 1ページPDFでは、サイト側の改ページ指定を一時的に抑制

## 使い方

1. Chromeで `chrome://extensions/` を開く
2. 右上の「デベロッパー モード」をオンにする
3. 「パッケージ化されていない拡張機能を読み込む」を押す
4. このフォルダ内の `extension` フォルダを選ぶ
5. PDF化したいページを開き、Pagefolio PDFのアイコンから保存する

## 注意点

- Chromeの高度なPDF機能を使うため、実行時にデバッグ接続の通知が出ます。
- `chrome://` やChromeウェブストアなど、一部のページはChromeの制約でPDF化できません。
- ログイン済みページは、開いているタブの状態を使ってPDF化します。
- PDF化は端末内で行い、外部サーバーへページ内容を送信しません。
- 非常に長いページは、1ページに収めるため自動で縮小します。
- Chromeが1ページとして扱える上限を超えるページは、1つのPDFファイル内でページ区切りに切り替えます。
- 保存先を選ばない設定にした場合は、Chromeのダウンロード設定に従って `Pagefolio PDF` フォルダへ保存します。
